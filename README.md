# ivan-kitchen-ai

一个厨房人理解 AI 的 Skill。

来自 3 个月、3253 对真实做菜问答的蒸馏。从「火烧到什么程度放鱼」出发，走到模型 → Agent → Skills → Harness Engineering。

## 用法

把 `SKILL.md` 里的内容加入任何 AI 的系统提示，它就知道：
- 厨房里哪些问题该问 AI
- 哪些场景你就是那个 AI 没有的信息
- 用厨房类比解释 AI 行业四层结构
